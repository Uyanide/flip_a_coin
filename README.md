# Wirf eine Münze

Dieses Projekt verwendet reine Windows-API und Gdiplus und kann mit CMake oder MSBuild gebaut werden. Alle Ressourcen sind über .rc-Dateien in das Projekt integriert, wodurch nur eine .exe-Datei benötigt wird, um es auszuführen.

Das Hauptziel des Projekts ist es, GIFs auf transparenten Fenstern mit einem transparenten Hintergrund abzuspielen. Die Implementierung erfolgt durch eine dedizierte Klasse, obwohl die Portabilität eingeschränkt sein kann.

## Tastaturbedienungen
| | |
| -------|-------|
| q | zum Ergebnis |
| r | wiederholen |
| space | pause |
| esc | beenden |
